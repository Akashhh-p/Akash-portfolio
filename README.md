# Akash Portfolio

Personal portfolio website for Akash Pentakota, an AI and ML engineer based in India at Centurion University.

The site highlights my work across machine learning, computer vision, generative AI, NLP, audio AI, and full-stack engineering. It includes selected projects, skills, certifications, internship experience, contact links, and interactive gamification features.

## Highlights

- Animated hero section with a 3D visual and a quick "Catch the Bug" challenge.
- Project showcase featuring Motion Guard AI, Multimodal Emotion Recognition, and AwakeMate.
- Skills grouped across ML, computer vision, Gen AI/NLP, multimodal AI, and engineering.
- Certifications and work sections for quick portfolio scanning.
- Gamified experience with XP, achievements, and interactive mini games.
- Responsive React interface built with TypeScript, Tailwind CSS, Framer Motion, and Lucide icons.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
- Supabase client

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
